Quem somos

Somos estudantes do curso de Tecnologia da Informação da Universidade

Virtual do Estado de São Paulo (UNIVESP), e esse é nosso trabalho do Projeto

Integrador I que tem como objetivo desenvolver um software com framework

web que utilize noções de banco de dados, praticando controle de versão.

Nesse site estamos utilizando a linguagem PHP, MySql para o banco de dados, linguagem

de marcação HTML, com controle de versão através do Git e com repositório

público no Github.
